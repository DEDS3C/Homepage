Logging in to the Router

To gain access to the wireless router’s configuration GUI, open a web browser. In the Address field, enter the default private IP address for your wireless router. The default IP address can be found in the documentation that came with the wireless router or you can search the Internet. The figure shows the IP address 192.168.0.1, which is a common default for some manufacturers. A security window prompts for authorization to access the router GUI. The word admin is commonly used as the default username and password. Again, check your wireless router’s documentation or search the Internet.

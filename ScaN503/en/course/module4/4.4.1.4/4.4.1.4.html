Configuring IP Addressing

In a home or small office network, you would normally leave the Internet connection set to automatic. However, for labs and Packet Tracer activities in this chapter, you will be connecting the wireless router to either a lab network or a simulated network. Internet service will not be enabled.

In Figure 1, the connection type is set to Static IP. The addressing parameters were manually configured. In Figure 2, the default DHCP addressing was changed to use the 10.10.10.0/24 subnet. The DNS address is set to the same DNS server as shown in Figure 1.

After this configuration is saved, you will lose connectivity to the wireless router. To regain access, renew your IP settings. Then enter the new router IP address, 10.10.10.1, in the address field of the web browser.

Planning a Wireless Router Implementation

Most home wireless routers are ready for service out of the box. They do not require any additional configuration. However, wireless router default IP addresses, usernames, and passwords can easily be found on the Internet. Just enter the search phrase “default wireless router ip address” or “default wireless router passwords” to see a listing of many websites that provide this information. Therefore, your first priority should be to change these defaults for security reasons.

Before installing a wireless router, consider changing the following default settings:

  • Internet Connection - IP address assignment for the Internet connection is normally set by the ISP through DHCP, as shown in the figure. However, you will statically configure this address in the Lab and Packet Tracer activities for this chapter.

  • DHCP Settings - Wireless routers come with DHCP already configured for the WLAN and LAN connections. However, it is a security best practice to change the default DHCP settings. These settings are part of the Network Setup shown in the figure.

  • SSID Name - The name of the WLAN network will be set to default name, which should be changed.

  • WLAN Security - Out of the box, home wireless routers will have no encryption and no security password. The WLAN will be open and available for any wireless device to connect. WPA2 with AES encryption and a strong password should be configured.

  • Management Access - The default passwords to access any brand of wireless router are readily available on the Internet. Therefore, the password should be changed to prevent unauthorized access to the wireless router’s configuration interface.

Video Demonstration - Data Loss and Retransmission

No matter how well designed a network is, data loss occasionally occurs; therefore, TCP provides methods of managing these segment losses. Among these is a mechanism to retransmit segments for unacknowledged data.

Note: Hosts today may also employ an optional feature called selective acknowledgment (SACK). If both hosts support SACK, it is possible for the destination to acknowledge bytes in discontinuous segments and the host would only need to retransmit the missing data.

Network protocols support the applications and services used by employees in a small network. Common network protocols are shown in the figure. Click each server for a brief description.

These network protocols comprise the fundamental toolset of a network professional. Each of these network protocols define:

  • Processes on either end of a communication session

  • Types of messages

  • Syntax of the messages

  • Meaning of informational fields

  • How messages are sent and the expected response

  • Interaction with the next lower layer

Many companies have established a policy of using secure versions of these protocols whenever possible. These protocols are HTTPS, SFTP, and SSH.

The figure is an interactive activity that shows several servers connected to a switch, which is connected to a router. When you click each server a callout shows a description of what the server does. The servers include DNS, Telnet, EMail, DHCP, Web, and FTP.

Bootset Files

Both Cisco routers and switches load the IOS image and startup configuration file into RAM when they are booted, as shown in the figure.

The running configuration is modified when the network administrator performs device configurations. Changes made to the running-config file should be saved to the startup configuration file in NVRAM, in case the router is restarted or loses power.

Figure shows a diagram of the files copied to RAM during bootup. From Flash, the IOS image file and other system related files are copied to RAM. From NVRAM, the startup-config is copied to RAM.

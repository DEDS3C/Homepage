The ipconfig Command

As shown in Figure 1, the IP address of the default gateway of a host can be viewed by issuing the ipconfig command at the command line of a Windows computer.

As shown in Figure 2, use the ipconfig /all command to view the MAC address, as well as a number of details regarding the Layer 3 addressing of the device.

The DNS Client service on Windows PCs also optimizes the performance of DNS name resolution by storing previously resolved names in memory. As shown in Figure 3, the ipconfig /displaydns command displays all of the cached DNS entries on a Windows computer system.

Figure 1 shows the output for the Windows command, “ipconfig”. The IP address, subnet mask, and default gateway are highlighted. Figure 2 shows the output for the Windows command, “ipconfig /all”. The Physical Address or MAC address is highlighted. Figure 3 shows the output for the Windows command, “ipconfig /displaydns”. The DNS information for cisco.com is shown.

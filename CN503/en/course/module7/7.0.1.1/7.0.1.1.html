Introduction

Security is a concern when using the public Internet to conduct business. Virtual Private Networks (VPNs) are used to ensure the security of data across the Internet. A VPN is used to create a private tunnel over a public network. Data can be secured by using encryption in this tunnel through the Internet and by using authentication to protect data from unauthorized access.

This chapter explains the concepts and processes related to VPNs, as well as the benefits of VPN implementations and the underlying protocols required to configure VPNs.
